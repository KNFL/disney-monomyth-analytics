# disney-monomyth-analytics
This project explores the Hero’s Journey structure in animated Disney films using data engineering and analytics techniques.

## Goals
- Build a reproducible data pipeline
- Extract narrative structure features
- Generate storytelling insights
- Deploy interactive dashboard

## Status
Project in early development phase.

## Project Structure
- data_raw/ → Raw collected data
- data_clean/ → Processed datasets
- notebooks/ → Exploratory analysis
- src/ → Production-ready scripts
- docs/ → Documentation and diagrams

## Tech Stack (Planned)
- Python
- Pandas
- SQL
- Airflow (future phase)
- Streamlit / Plotly
